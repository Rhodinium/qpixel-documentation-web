## Feature:

- Support MLO & Shell
- Simple UI
- Player or job can own motel
- Police can raid the motel
- Run at 0.00 ms - 0.05 ms max
- Motel key as an item
- Anyone that have key can open the motel
- You can reset the door, incase if your motel key is stolen
